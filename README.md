# Vite + React Random Tone & Quiz App

This project is a React app scaffolded with Vite. It features:

- A sidebar menu for navigation
- A Random Tone generator (fully implemented)
- A Quiz page (to be implemented)

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Features

- **Random Tone**: Play random musical tones and configure duration, note range, and half tones.
- **Quiz**: Placeholder for future quiz functionality.

## Keyboard Shortcuts

- **Spacebar**: Repeat last tone
- **Enter**: Play a new random tone

## Project Structure

- `src/RandomToneApp.jsx`: Random Tone generator component
- `src/Quiz.jsx`: Quiz page (empty)
- `src/App.jsx`: Main app with sidebar navigation
